# Homebrew Tap for dev-cli

This tap contains the Homebrew formula for [dev-cli](https://github.com/AlpineJosh/dev-cli), a Git worktree and project management CLI.

## Installation

```bash
brew tap AlpineJosh/tap
brew install dev
```

## Upgrade

```bash
brew upgrade dev
```

## Uninstall

```bash
brew uninstall dev
brew untap AlpineJosh/tap
```
